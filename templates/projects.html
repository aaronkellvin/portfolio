Maritime Vessel Inventory System

The Maritime Vessel Inventory System (MVIS) is a comprehensive inventory and maintenance management system developed during my internship at the Philippine National Police (PNP) Maritime Group. The system was designed to monitor and manage vessel parts, equipment, and Preventive Maintenance Services (PMS) across maritime units and regional offices nationwide.

MVIS centralizes inventory records, streamlines asset tracking, and monitors maintenance schedules and repair activities to ensure operational readiness. I contributed the system's interactive maritime map built with SVG and designed the frontend features for each page, giving users a clear, consistent interface to explore units and vessel resources by region while supporting accurate reporting and real-time operational visibility.

Through this project, I gained valuable experience in full-stack web development, frontend design, database design, system analysis, SVG-based map development, and the development of enterprise-level solutions that support large-scale operational processes.

Technologies Used: Python, Flask, MySQL, HTML, CSS, JavaScript, SVG

Role: Full-stack Web Developer

PRS Memo Tracking System

The PRS Memo Tracking System is a web-based document management and tracking solution developed for the Philippine National Police (PNP) Maritime Group. The system was designed to streamline the distribution, monitoring, and management of internal memoranda across departments and offices.

By centralizing memo records and automating tracking processes, the system improves document visibility, enhances information accuracy, and enables efficient monitoring of memo status throughout its lifecycle. I designed the frontend features for each page of the application, creating intuitive layouts that help users track, review, and manage memoranda with less friction.

Through this project, I strengthened my skills in full-stack web development, frontend design, database management, system design, and the development of solutions that improve organizational efficiency and document control processes.

Technologies Used: Python, Flask, MySQL, HTML, CSS, JavaScript

Role: Full-stack Web Developer

AI-Driven Web-Based Transaction Processing System for ISO Documentation with Automated Content Evaluation and Redundancy Detection in Letran Calamba

This capstone project is a web-based transaction processing system designed to automate ISO 21001:2025 documentation workflows at Colegio de San Juan de Letran Calamba. The system streamlines document management through role-based access control, user authentication, and secure electronic signatures supported by hashing, digital certificates, and verified signing workflows.

The platform uses AI to perform content evaluation on each uploaded document and detect redundancies across the documentation library, helping improve quality, consistency, and ISO compliance. I designed the frontend features for each page of the system, ensuring role-based workflows, document views, and approval screens remained clear, responsive, and easy to navigate.

Through this project, I applied my skills in full-stack web development, frontend design, database design, system analysis, authentication and digital signature security, artificial intelligence integration, and software engineering to develop a solution that addresses real-world organizational and compliance challenges.

Technologies Used: Python, Flask, MySQL, HTML, CSS, JavaScript

Security Features: Authentication, hashing for digital signatures, digital certificates

AI Features: Content evaluation for uploaded documents, redundancy detection

AI Technologies Used: OpenAI, GPT-4, GPT-4o

Hosting: Hostinger

Role: Full-stack Web Developer

KIOSK Machine Project

The KIOSK Machine Project is an interactive self-service food ordering application developed using Java Swing and MySQL. Knightbot — ISO Documentation Assistant

Knightbot is an AI-powered in-system guide built for the ISO Documentation platform at Colegio de San Juan de Letran Calamba. It provides real-time assistance to users, administrators, and the final approver on how to navigate and use the system effectively.

The chatbot utilizes artificial intelligence to respond to questions and walk users through each page step by step, helping them complete tasks correctly and smoothly for a more efficient experience. I also designed the frontend features of the chat interface and supporting UI elements so guidance felt natural within each page of the ISO Documentation platform.

Knightbot was integrated as a guide within the AI-Driven Web-Based Transaction Processing System for ISO Documentation with Automated Content Evaluation and Redundancy Detection in Letran Calamba, improving usability and reducing the learning curve for every role in the system.

Ozzy Party Nest

Ozzy Party Nest is an individual web-based booking platform developed for a Sydney-based event rental business. The system enables customers to easily browse and book event-related services, including tables and chairs rental, professional sound system setup, and van rental with guest transportation services.

The website was developed using Python Flask for the backend and HTML, CSS, and JavaScript for the frontend and application functionality. I designed the frontend features for each page, including the service catalog, booking flow, and contact sections, with a modern responsive layout that allows customers to submit inquiries and reservations online.

The platform helps streamline the booking process, improve customer engagement, and provide a convenient digital presence for the business across Sydney, Australia.
